# 💰 Smart Expense Tracker

A Python-based application that helps users track expenses, analyze spending patterns, and predict future savings.

## 🚀 Features
- 📊 Weekly expense tracking
- 📈 Spending prediction (Average + Trend based)
- 🎯 Goal management system
- 💡 Financial advice system
- 🔮 What-if simulator
- 📉 Graph visualization

## 🛠️ Technologies Used
- Python
- Matplotlib
- JSON (for data storage)

## ▶️ How to Run
```bash
python main.py
